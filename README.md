# Mi Portafolio Profesional

¡Bienvenido! Este es el código fuente de mi portafolio personal, donde muestro mi trabajo como Desarrollador Web Front End.

## Tecnologías utilizadas
* HTML5 y CSS3 (Diseño responsivo y custom properties)
* JavaScript Vanilla
* FontAwesome para iconos
* Git y GitHub para control de versiones

## Estructura del proyecto
* `css/`: Estilos organizados por secciones.
* `assets/`: Imágenes y recursos visuales.
* `index.html`: Estructura principal.
